# 📂 Reports Directory

이 폴더는 **'주택가격 예측을 통한 비즈니스 제안'** 프로젝트의 최종 결과물과 데이터 인사이트를 정리한 보고서가 보관되는 공간입니다.

---

## 📑 주요 포함 파일
현재 폴더에는 다음과 같은 핵심 자료들이 포함되어 있습니다.

* **`최종.pdf`**: 프로젝트의 전체 흐름(의문점-가설-분석-결론)을 담은 최종 발표 자료입니다.
* **시각화 결과물**: 주요 데이터 분석 차트 및 그래프 이미지(PNG/JPG)가 포함될 수 있습니다.

---

## 💡 참고 사항
* 모든 시각화 그래프의 원천 코드는 상위 디렉토리의 `source/` 폴더 내 파이썬 스크립트를 통해 확인 가능합니다.
* 분석에 사용된 기초 통계 데이터는 `datasets/` 폴더를 참조해 주세요.
